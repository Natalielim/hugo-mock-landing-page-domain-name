### Local Development
1. Install Hugo (extended version)
bash
brew install hugo
2. Clone the repository
git clone [[repository-url](https://github.com/Natalielim/hugo-mock-landing-page.git)]
cd [repository-name]hugo-mock-landing-page
3. Start the Hugo server
hugo server
